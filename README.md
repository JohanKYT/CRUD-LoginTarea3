##  Descripción del Proyecto

Este proyecto es una aplicación web backend desarrollada en **Java y Spring Boot** que implementa el patrón de arquitectura **MVC (Modelo-Vista-Controlador)**. Su objetivo principal es gestionar una base de datos de usuarios a través de operaciones CRUD completas, todo esto protegido por un sistema de autenticación sólido. 

Las rutas están aseguradas para que ningún usuario anónimo pueda acceder a la información sin antes validar sus credenciales.

##  Características y Demostración

- `Autenticación (Login):` Pantalla de inicio de sesión validada y segura. Las sesiones no autorizadas son redirigidas automáticamente.
- `Operaciones CRUD:`
  - **Create:** Formulario para registrar nuevos usuarios.
  - **Read:** Tabla dinámica que lista todos los usuarios registrados.
  - **Update:** Opción para editar la información (Nombre y Email) de un usuario existente mediante su ID.
  - **Delete:** Eliminación rápida de registros de la base de datos.
- `Diseño Responsivo:` Interfaz limpia y adaptativa construida con **CSS Grid** puro y HTML semántico, sin depender de librerías externas.
